# 🎸 RockWear - Landing Page de Poleras de Bandas de Rock

![RockWear Banner](https://placehold.co/1920x400/1a1a1a/ffffff?text=RockWear+-+Viste+tu+ROCK)

## 📋 Descripción del Proyecto

RockWear es una landing page moderna y atractiva diseñada para la venta de poleras oficiales de bandas de rock legendarias. El sitio presenta un diseño oscuro y llamativo que refleja la esencia del rock, con cards interactivas, sistema de carrito simulado, formulario de contacto y una experiencia de usuario fluida.

### 🎯 Propósito del Proyecto

Este proyecto fue desarrollado como **Evaluación Sumativa 1** para demostrar competencias en:
- HTML5 semántico
- CSS3 con diseño responsivo
- JavaScript para interactividad
- Bootstrap 5
- Control de versiones con Git

---

## 🚀 Características Principales

### ✨ Funcionalidades
- ✅ **Hero section** con llamado a la acción
- ✅ **Cards de productos** con información de poleras de bandas

- ✅ **Notificaciones toast** para feedback visual
- ✅ **Formulario de contacto** con validación
- ✅ **Diseño completamente responsivo** (mobile-first)

## 📱 Responsividad

El sitio está optimizado para los siguientes breakpoints:

|-------------|-------|---------|
| 📱 Móvil | < 768px | Textos reducidos, cards apiladas |
